仓廪实而知礼节，衣食足而知荣辱

Posted by Chen on 2017-05-29

说这么久我只能呵呵
本来不想回你的，等你说，你倒越来越起劲了，杨梁给你说了什么我不清楚，
但在高一的时候我就知道他给你说完了，他算个什么？打架有多nb？我不见得他家有多diao，叫人的话我也不觉得我叫的人不多。高一刚开学的时候我拿王卓手机那件事情，这个事情也能算你威胁人的一种把柄我觉得也是够了，我是不是他说的那样的人你不知道，你也不会知道，不知道他说没说，那我今天就把那件事情说出来。

手机一事

他说我偷手机也好，还是怎么的，我当时缺手机用？只不过当时想在他手机上装监听软件而已。

当时它放在寝室充电，我要弄那东西一弄就是一两个小时，他们回来的时候已经晚了，但是我还回去照样得误会
与其这样被误会，为何赌一赌不让他失踪呢？ 他们在寝室找，我还是没还。不过这种事情藏不了一辈子，我并不稀奇那个手机，我也没带回家，而是放在寝室，因为我知道我根本没有做那样的事，结果呢，还是被发现了，这种事情被他们发现了该怎么解决呢？当然是挨打，最后挨了打就完事了。然后杨梁吹逼他们有多牛，被税森林按在地上摩擦的人，给你吹他们有多叼…..我可以说就算在江油打架，怕是叫的人都没我多，他算个啥？就是在学校和几个认识的人，就是很刁的人了？ 这件事情我姑爷是知道的，本来说要找到学校找到江油去打回来，被我否决了。这种事情就是自己错在先嘛！

不管你如何如何的认为我是一个如何如何的人，我做好了我自己

回应

  1. 悄咪咪删好友，怂了？

    我删好友还给您贵人通知一声？

  2. 你说的我和你们玩游戏的时候高冷

    请问你和文聊天的时候，还知道有我在吗？你们聊你们的我插不上嘴这也算是高冷？

  3. 一句腊鸡就把我惹到了？

    没有，用你的话来说：”就是看你不爽”

  4. 你看不惯就能骂，别人看不惯你就不行了？

    大贵人你这是把你自己看得得有多高？ 不回你只是不想把话说绝了。其他的话不想多说

  5. 怂到发空间博取同情？

    不知道您哪只眼睛看见我发空间博取同情来的，我空间一直开放，还怕你看？

什么 只要你说话我就骂你爹骂你妈，一天烂逼都给你装完了(没能装上逼怪我咯)

其次，你非要说我装逼的话，我也没办法，其实就是你的无能吧，谁都会”装逼”，你我是同行就不这样觉得了。装逼这个东西的标准是什么？你知道吗？在一些人眼里有时候你一个无意的小小的动作在别人眼里都是装逼！我还能说什么？装逼的意义在于，输人不输脸面。装逼有两种，一种是成功，一种是失败。成功暂且不论，失败的意义是教你做人，认清自己，打破不切实际的幻想。换一句话说就是装逼靠实力，不靠嘴皮！只有把学习当做装逼的主战场，带着一股谜一样的自信去强势装逼，才能促进学习。在心理学上为什么不会有“装逼心理学”这种粗俗的称谓，而是叫做：成就目标定向理论呢？

总结

本来想把你屏蔽了，但我觉得没有那个必要

自己不讲理的一嘴脏话，耍了流氓还非得人家回复你你才满足，怕对我的伤害不够量吗？给你说我全都看见了，这下你满足了？我的内心毫无波动甚至还想笑。你的话就是为了衬托最后那句 “我话费多，我很会说，要撕逼打电话，还是长途的，就问你敢不敢，不敢就是怂了”

一张嘴就是一个妈一张嘴就是一口城乡结合部的气息，还骂出自信了。自己缺少对世界现实性的认知，其视野严重受到局限，缺少兼容并蓄的接受和理解他人思想的能力。同时，又倾向于把自己对世界的理解做为衡量他人言行的标准，所以，对于你这样的人来讲，当他对别人的言论大加批判时，在他的内心，他会将自己的言论感知为正确的，或者是善意的。而他的内心，可能又缺少共情性的去理解他人的能力，所以，他也就不会感受到自己的言行是在对他人造成伤害，坚持认为自己是一个有理想有抱负的人，一个脱离了低级趣味的人，一个坚持真理的人。几乎没说一句话都觉得自己是在替天行道。我都这样说了还要让我回复你？

综上所诉，对你的个人评价： 喷子

用狗来举例好了——他们，只能算是狗头人

狗头人，不是狗。他们可以理解一部分的狗的思路，甚至用狗的语言去跟狗交谈，但他们永远无法探寻狗的意识和狗的三观。

狗的语言，是非常匮乏的。他们的语言里只有“你妈那批”和“日吗”之类的语气词，勉强表达一下心情和状态还可以，但想要透露出“我其实知道自己是一条狗，但我不以自己为耻——事实上这样的人生才是我想要的”这么复杂的内容，就根本不可能。这么说我就太埋汰狗了，其实狗比你都要强多了。

  • 好比如被对方某句话挑起的攻击性，比如被他人表现出来的优越感产生的憎恨，比如莫名而来的嫉妒，比如本来在生活中就积累了足够多的戾气……所有的这些，都可以让你被称作“喷子”，虽然也具备换位思考的能力，却选择了不那么做，因为一旦承认自己的偏颇，接纳了对方观点，就等于“认输”。

  • 断章取义，他只看你有没有留下漏洞，把柄。只要有就抓出来狂批狠斗。

  • 出口成脏，一般都说不出什么好话，语言能有多恶毒就有多恶毒，以激怒对方为辅助，挑起争端为目的。

  • 自我感觉良好，以挑刺找茬为乐趣，通过发表反对观点进行自我实现

  • 依靠狐朋狗友，以多喷寡 (这个暂且不论)
